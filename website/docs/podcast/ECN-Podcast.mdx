# ECN Podcast

ECN podcast 是一档专注于与区块链从业者深入对话，探讨以太坊和泛区块链技术以及生态发展的播客。我们期待理性的对话能激起听众的思考、真诚的沟通能让 web3 更有温度。
﹋﹊﹋﹊﹋﹊﹋﹊﹋﹊﹋﹊﹋﹊﹋﹊﹋﹊
我们是谁：
ECN (Ethereum Community Network) 自创立以来就一直致力于以太坊知识普及与教育，目标是让更广泛的人群了解和使用以太坊以及区块链。ECN 正在探索一种与社区更紧密的互动方式，我们希望通过 Podcast 的方式将更多 Web3 背后的有趣观点传递给大众。
﹋﹊﹋﹊﹋﹊﹋﹊﹋﹊﹋﹊﹋﹊﹋﹊﹋﹊
我们正在构建：
EthGifts -- 这是一个用于发布各种数字凭证的平台 (灵魂绑定代币/SBT | 可验证凭证/Verifiable Credentials | NFT)，通过此平台对 ECN 社区成员的参与以及贡献进行编码。ECN 社区成员可以在 EthGifts 上查看他们的参与记录，然后铸造 SBT/NFT 或者下载 VC。STAY TUNE!

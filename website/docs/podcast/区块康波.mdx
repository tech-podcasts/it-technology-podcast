# 区块康波

区块康波是一档由ColinLy与StevenZ创办的播客节目。
本节目将重点关注区块链行业最新的发展动态，挖掘有潜力穿越牛熊周期的区块链项目，并通过与行业从业者的深度交流，探索区块链行业未来的发展趋势。
希望我们能够通过这档节目，与听众一起“穿越周期，看见未来”！

ColinLy，加密行业研究员，研究方向包括：衍生品、算法稳定币、NFT Marketplace、X to Earn。推特ID：@ly_wander；公众号：佳霖的区块链笔记
StevenZ，加密行业深度文章作者，全能型研究员。推特ID：@_0x76；公众号：76er

# Founder 100

Founder Park 是 GeekPark（极客公园）旗下的创始人社区，聚集着一批关心「早期科技创业」的创业者/投资人/合作伙伴...

为了让「有价值的认知」流动起来，我们在小宇宙上做了一档节目【Founder 100】，希望能带你认识 100 位有潜力的科技创业者👊欢迎订阅🎉

👉在这里「改变世界的想法」或许正在诞生
- 虚拟人、游戏、社交...元宇宙的想象没有天花板
- 钢铁侠并非科幻，外骨骼已经开始成为新工具
- 仓储、物流、制造的供应链，如何进行数字化改造？
- 商业航天中，有人想把「数据中心」搬到太空
- 有人坚信「量子计算」才是算力的终极形态

持续收听，咱们共同见证科技创新的力量～

# 创新驱动力

分享万众创新的路上，来自创业者和专业人士的创新故事。
听他们聊聊技术是如何引领各行各业的变革、转型，请他们深度解读不同领域的创新产品与商业模式，洞察新技术、新商业如何成为当今社会经济的驱动力。
当然还有这些创新者的心路历程，探寻他们内心的驱动力。

# 先见未明

先见，必然未明。创新路上，我们一同求索。
-
当人们说“某人真有先见之明”时，它本身就已经是“后见之明”。

任何创新，无论技术还是商业，都必须经过验证。验证的过程，往往是漫长、混乱、挣扎，充满失败的——甚至，验证的结果也可能是被证伪。

幸运的是，自人类在地球上扎根至今，创意不息，创新不止。

⌈先见未明⌋ 希望与你一起，发掘和记录生生不息的科技创新，探索创新背后的真价值，不人云亦云，不后见之明。

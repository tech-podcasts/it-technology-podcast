# Web3 101

也许是一片新大陆，但也是荒野的西部，Web3 的世界里弥漫着去中心化的福音，呼啸着庞氏陷阱的警告。未知而且粗糙，却吸引越来越多的开发者进入。《Web3 101》是一档探索 Web3 世界的播客，在这个技术与商业上采用了全新配比的世界里，跟着Jane与阿伟展开一趟冒险之旅。

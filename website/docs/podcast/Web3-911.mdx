# Web3 911

这是一档通过和Web3一线从业者、参与者对话的形式，展现Web3最新发展情况的节目。

节目内容，将主要聚焦以太坊 Ethereum 生态发展，展现每个项目的独特风采，致力于将复杂难懂的技术细节抽象成通俗易懂的应用场景，让更多Web2的开发者和用户，更快更容易的了解到以太坊的巨大发展空间和生态样貌。

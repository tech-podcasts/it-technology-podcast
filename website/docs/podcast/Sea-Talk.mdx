# Sea Talk

Sea Talk 是一档访谈对话类 Web3 播客节目，以人为本，每周邀请 Web3 局内人和从业者聊天，挖掘他们关于技术、产品与投资的洞察，传播共识与观点。

主理人 Sea 在做了十多年 Web2 产品经理和创业者后，一头扎进 Web3 兔子洞。

# 科技双眼啤

「佐声道」出品的科技双眼啤，由开发者Cali与设计师Rather共同主持，我们是从设计到开发等不同的视角对互联网软件硬件进行深入探讨的科技播客。

佐声道主理人 Cali 在留美期间有丰富的工作经验，对于产品有极度的热爱以及不同的体验经历； 设计师 Rather 一直在国内从事着UI设计师的工作，并在这一行有着不同的独特见解。 同为苹果粉的两人一拍即合，他们也将自己对于产品的解读认知、对设计的严谨态度都延续到了他们所有的节目当中。

# STC小蛮邀

小蛮腰科技大会是由广州市科技局、亚洲数据集团共同主办的“科技+”盛会，用智慧与勇气探索未知世界。大会依托人工智能、数字经济，探讨智能制造、创业投资、元宇宙和商业航天、新消费等多个议题，邀请全球顶尖技术研发者、专家教授、企业家、资本代表、媒体人，通过主题演讲、圆桌对话、闭门会议、展览展示等形式探讨国内外的科技新进程，探索人类文明新进展。
过去10年间，有小米科技雷军、中欧商学院许小年、中国社科院樊纲、IDG资本熊晓鸽、软银亚洲阎焱、红杉资本周逵等近千位嘉宾登上大会舞台，向世界分享了他们对于科技世界的远见卓识。

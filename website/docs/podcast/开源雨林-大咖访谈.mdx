# 开源雨林—大咖访谈

每期节目时长30-45分钟，希望能通过与相关领域专业人士的真实对话，能给身处开源圈子的你带来一些思考与启迪。同时也想借助这档节目，把开源背后的逻辑、故事，用不高深、不枯燥的语言，分享给对开源充满好奇的听众，在“闲聊"氛围中获得密集的知识养料，通过声音认识更丰富的开源世界。

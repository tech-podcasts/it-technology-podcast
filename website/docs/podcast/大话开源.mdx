# 大话开源

「大话开源」是一档由 OpenTEKr 发起的访谈节目，专注于沉淀开源人的所思所行，力求摸索出有趣、有料、有品的「开源武林秘笈」。 每一期对话，我们会邀请身经百战的开源老兵，或冉冉升起的开源之星，来分享他们的前沿洞察、防坑指南，以及鲜为人知的走心故事。在这里，不管您是“开源发烧友”，还是“门外驻足观望的好奇宝宝”，都可以汲取与开源相关的更多心法与招数。

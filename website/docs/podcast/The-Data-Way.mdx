# The Data Way

【The Data Way】是由 SphereEx 公司出品的播客节目，这里有开源、数据、技术的故事。同时，我们关注开发者的工作日常，也讨论开发者的生活日常。我们聚焦开源，也鼓励开发者积极拥抱开源以外的多彩生活。这些，都将在【The Data Way】的节目中一一得到展现。 

目前【The Data Way】已上线各大音频平台，每两周的周一更新，欢迎大家积极点击订阅，第一时间接收【The Data Way】的更新消息。

# Go 夜聊

「Go 夜聊」是一档由杨文和欧长坤主持的针对 Go 语言发展史的播客节目。我们的宗旨是让更多的人了解 Go 语言的发展历史和未来规划，希望大家在这里可以了解到 Go 语言的方方面面。如果你觉得「Go 夜聊」做的还不错，欢迎你推荐给身边的朋友。

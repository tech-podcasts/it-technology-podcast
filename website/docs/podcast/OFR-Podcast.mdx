# OFR Podcast

关于 OFR
OFR 是一家多策略的区块链投资基金，团队成员由前币安高管和投资团队于 2021 年 7 月创立，创始合伙人包括前币安战略投资并购副总裁 Ling、前企业发展部负责人 Wayne、前 Binance Labs 投资总监和 Binance Launchpad 负责人JX。

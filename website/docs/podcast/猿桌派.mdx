# 猿桌派

首档程序员谈话综艺

把一线程序员请上桌，听他们自己说！​

节目由融云携手 CSDN 联合打造，邀请多位在相关技术上积累多年、在职业发展上过关斩将的一线程序员，一起“聊工作聊人生，从前沿技术谈到职业成长”。

节目设置“猿代码”、“猿生活”两个主要环节，除了不容错过的技术干货，还将从各位嘉宾的职业生涯、技术进阶、成长焦虑等亲身经历出发，与大家吐槽嗨聊、真诚交流，期待为年轻程序员带来有价值的职业规划参考，与资深程序员共同探寻职业瓶颈期的破局之法。

# Nebula Graph 星球

我们是 Nebula 社区的布道者，专注于开源分布式图数据库的技术分享。这个频道会带你了解到 Nebula Graph 每一次发版的新增功能和优化点的细节介绍，还有他们的使用场景等相关信息。 我们希望更多图数据库的爱好者参与到我们这个开源的项目里，和我们一起学习讨论关于图数据库的发展和技术细节的探讨...

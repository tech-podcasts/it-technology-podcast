# KubeSphere Talk

KubeSphere Talk 是由 KubeSphere 发起的一档访谈、聊天的音频节目。节目前期，会主要邀请 KubeSphere 社区的 Ambassador、Contributor 等角色，围绕云原生、开源等热门话题展开，挖掘他们身上的故事和闪光点，为期望了解开源和云原生的各位小伙伴提供参考。
